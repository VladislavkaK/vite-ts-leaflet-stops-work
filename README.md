# Map with markers

Нужно отобразить карту используя данные которые находятся в `/public/data`.

Использовал для решения, следующие технолгии:

1) React
2) Vite
3) Typescript
4) Redux
5) Redux-toolkit
6) Leaflet
7) Module styles (+ sass)
8) Yandex Maps API
